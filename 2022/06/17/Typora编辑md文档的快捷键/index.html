    Typora编辑md文档的快捷键

    F9打字机模式

    F8专注模式

    shift+F12开发者模式

    ctrl+/源代码模式

    ctrl+tab应用内窗口切换

    ctrl+shift+f搜索 F3下一个

    1、写标题

    ctrl+1表示一级标题

    ctrl+2表示二级标题

    以此类推。

    2、分割线

    使用三个或以上的 — 或者 * 表示，且这一行只有符号，注意不要被识别为二级标题即可（可采用中间或者前面可以加空格）。

    3、斜体和粗体

    使用 * 和 ** 分别表示斜体和粗体，删除线使用两个~ 表示。

    斜体快捷键Ctrl+I

    粗体快捷键Ctrl+B

    4、超链接和图片

    超链接ctrl+K

    插入图片ctrl+shift+I

    链接和图片的写法类似，图片仅在超链接前多一个！，一般是[文字描述]（链接）

    5、无序列表ctrl+shift+]

    使用 —、+ 和 * 表示无序列表， 前后留一行空白，可嵌套

    6、有序列表ctrl+shift+[

    使用1. （点号后面有个空格）表示有序列表，可嵌套。

    7、文字引用

    引用ctrl+shift+Q

    使用 > 表示，可以有多个 > ,表示层级更深。

    8、行内代码块

    使用 ` 表示。转义使用反斜杠 \ 进行转义

    9、代码块ctrl+shift+k

    使用四个空格缩进表示代码块，一些IDE支持行数提示和着色，一般使用三个 ` 表示。

    10、表格

    ctrl+T 生成表格。

    直接看例子

    “|商品|数量|单价|”

    输入以上即可得到下方表格，默认左对齐 ，|—：|表示右对齐，|：—：|表示居中对齐。

    商品 数量 单价
    苹果 10 1

    11、流程图

    主要的语法为name=> type: describe, 其中type 主要有以下几种：

    (1)、开始和结束：start end

    (2)、输入和输出：input output

    (3)、操作： operation

    (4)、条件：condition

    (5)、子程序：subroutine

    st=>start: Start:>https://www.zybuluo.com
io=>inputoutput: verification
op=>operation: Your Operaiton
cond=>condition: yes or No?
sub=>subroutine: Your Subroutine 
e=>end 

st->io->op->cond
cond(yes)->e
cond(no)->sub->io

e=>end

    12、数学公式ctrl+shift+m

    使用$表示，其中一个$ 表示在行内，两个$表示独占一行。

    支持 LaTeX 编辑显示支持，访问 MathJax 参考更多使用方法。

    推荐一个常用的数学公式在线编译网站：https://private.codecogs.com/latex/eqneditor.php

    13、支持HTML标签

    (1)、例如想要段落的缩进，可以如下：

    &nbsp;&nbsp;不断行的空白格&nbsp;或者&#160;
&ensp:&ensp;半方大的空白&ensp;或者&#8194；
&emsp;&emsp:全方大的空白&emsp;或者&#8195；

    &nbsp;&nbsp;不断行的空白格&nbsp;或者&#160;
    &ensp;&ensp;半方大的空白&ensp;或者&#8194;
    &emsp;&emsp;全方大的空白&emsp;或者&#8195;

    (2)、页面内的跳转

    14、思维导图

    输入```mermaid，即可初始化一张空白图。

    流程图实例：

    语法解释TD即自上而下，LR即自左向右。

    graph TD;
    内容-->核心;
    内容-->标题;
    核心-->注意点;

    效果：

    graph TD;
    内容-->核心;
    内容-->标题;
    核心-->注意点;

    时序图实例：

    语法解释：->>代表实线，- ->>则代表虚线。

    sequenceDiagram
    Alice->>John: hello John，how are you?
    John-->>Alice: Great!

    效果：

    sequenceDiagram
    Alice->>John: Hello John, how are you?
    John-->>Alice: Great!

    状态图实例：

    语法解释：[*]表示开始或者结束，如果在箭头右边则表示结束；

    stateDiagram
    [*]-->s1
    s1-->[*]

    效果：

    stateDiagram
    [*]-->s1
    s1-->[*]

    类图：

    语法解释：<|– 表示继承，+表示public，-表示private;

    classDiagram
    Animal <|-- Duck
    Animal <|-- Fish
    Animal <|-- Zebra
    Animal : +int age
    Animal : +String gender
    Animal : +isMammal()
    Animal：+mate()
    class Duck{
        +String beakColor
        +swim()
        +quack()
        }
    class Fish{
        -int sizeInFeet
        -canEat()
        }
    class Zebra{
        +bool is_wild
        +run()
        }

    甘特图：

    甘特图一般用来表示项目的计划排期，目前在工作中经常会用到。

    语法也非常简单，从上到下依次是图片标题，日期格式，项目，项目细分的任务。

    gantt
    title 工作计划
    dateFormat YYYY-MM-DD
    section Section
    A task  :a1, 2020-01-01,30d
    Another task :after a1, 20d
    section Another
    Task in sec  :2020-01-12, 12d
    another task :24d

    效果

    gantt
    title 工作计划
    dateFormat YYYY-MM-DD
    section Section
    A task :a1, 2020-01-01,30d
    Another task :after a1, 20d
    section Another
    Task in sec :2020-01-12,12d
    another task: 24d
